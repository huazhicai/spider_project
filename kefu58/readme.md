### 目标
- 爬取58同城上，杭州市，淘宝/天猫/京东 等电商客服的招聘企业信息
- 信息存入mysql数据库

### 环境
- Python 3.6.1
- pymysql
- scrapy 1.0
- scrapy-redis

### 设计思路
- 起始url:http://hz.58.com/job/?key=客服&final=1&jump=1
- Rule()方式提取链接，做全站爬取
- 在索引页正则匹`[淘宝|天猫|京东|速卖通|亚马逊|电商|EBAY|唯品会]*?`
提取目标公司url， 进入到目标公司详情页，提取需要的信息
- 有一部分电话号码是图片格式，需要用到tesstor识别图片上的文字
- 爬虫可能会要限制Ip的情况。 爬取过程要认为减速。
